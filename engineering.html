Web制作・開発

対応内容

  • 静的Webサイト制作（GitHub Pages 等）
  • 個人事業・ポートフォリオサイト構築
  • HTML / CSS / JavaScript
  • microCMS を用いた更新可能なWebサイト
  • 小規模Web改善・改修

制作の流れ

  1. Google Formより、ご依頼内容やご要望をヒアリングいたします。
  2. 個人情報の取り扱いに関する同意書（電子署名）をご確認・ご提出いただきます。
  3. ご要望をもとに、簡易的なイメージページをご提案いたします。
  4. 内容確定後、お見積もりを作成し、契約書にご署名・着手金（30％）をお支払いいただきます。
  5. 入金確認後に制作を開始し、完成後に残金（70％）のお支払いをもって納品となります。

修正回数について

制作中の修正は 最大5回まで 無償で対応いたします。

修正の質を保ち、すべてのご依頼者さまに公平な制作時間を確保するため、
修正回数に上限を設けています。

際限なく修正を重ねてしまうと制作スケジュールが不安定になり、
他のご依頼をお受けできなくなるリスクが生じてしまいます。

そのため、各修正ごとに内容をまとめてご共有いただく形をお願いしております。

※6回目以降の修正、または大幅な方向転換については、
追加費用をご相談させていただく場合があります。

運用・保守について

サイト公開後の運用・保守についても対応可能です。

  • テキスト・画像の差し替え
  • Topics / News の更新サポート
  • 軽微なレイアウト調整
  • 表示崩れ・不具合対応

内容や頻度に応じて、
月額運用契約（例：¥5,000 / 月〜） にて対応いたします。

こんな方に

技術的な説明が分からなくても問題ありません。
ご要望をヒアリングし、目的に合ったWebサイトをご提案します。
